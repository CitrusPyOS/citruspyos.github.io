CitrusPy OS

A fast, lightweight, and open-source operating system built for efficiency and simplicity.

Features

New

CitrusPy OS is a completely new approach to operating systems with lots of exclusive functions.

Open-Source

As an open-source project, CitrusPy OS is fully transparent, customizable, and free to use.

Command Line

The command line interface is easy to use and helps you stay organised in comparison to multi tasking systems.

Get Started

Download the latest version of CitrusPy OS and start experiencing a new way of computing.
